Data interpretation :
BCD format

msg[0] = sequence code
msg[1..3] = sensor_id
msg[4] = channel
msg[5..6] = rolling code
msg[7] = flag (0: temperature, 2: clock, 8: not paired, 4: battery low?)
msg[8..n] = data
msg[n-4..n-3] postamble ? seems to be related to sequence code
msg[n-2..n-1] checksum ?
